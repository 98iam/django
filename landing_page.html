Got Questions?

Frequently Asked Questions

Is InvManage Pro free to use?
Currently, InvManage Pro offers a generous free tier perfect for small businesses and individuals. We may introduce optional paid plans with advanced features in the future. For now, enjoy the core functionality for free!
What do I need to run this system?
You'll need Python (3.8+), PostgreSQL (or configure SQLite for development), and Git. An optional Google Gemini API Key is required for the AI Chat Assistant. Detailed setup instructions are in the README on GitHub.
Is my data secure?
Yes. We utilize Django's robust security features. If using Supabase, you benefit from their enterprise-grade security. Multi-user support includes data isolation, meaning users can only access their own inventory data. The AI assistant also respects this data isolation.
Can I contribute to the project?
Absolutely! Pull requests are welcome. Please check the `CONTRIBUTING.md` file (if available) or open an issue on GitHub first to discuss major changes. We appreciate community involvement.
